Custom subscribe plugin

This plugin is a custom wordpress plugin for subscriptions with several functionalities. You have evidence of new subscribers, date of subscriptions, several options to customize subscribe button, subscription email, confirmation email etc.


INSTALLATION

1. Upload wp-subscribe-plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Subscribe Widget to one of your sidebars.
4. You are good to go.
